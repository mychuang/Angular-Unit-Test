# Ng test report

- 不顯示測試結果, 結果產在/coverage資料夾

  ```
  ng test --no-watch --code-coverage
  ```

- 顯示測試結果, 結果同時產在/coverage資料夾

  ```
  ng test --code-coverage
  ```
